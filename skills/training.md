# Training Programming

## Current Context
- Dave prefers weight training
- Gym closes 9:30pm
- Gaming causes chronic fatigue in right shoulder, forearm, and hand
- Push/pull sessions compete with gaming recovery
- Legs do NOT interfere with gaming — can be trained hard and frequently
- Arms are the visual priority (t-shirt muscles)
- Muscle memory is active — Dave previously trained to significant strength levels

## Programming Principles
- High-frequency arm training (3-4x/week) is optimal for his goal
- Light pump work (15-20 reps) aids recovery via blood flow without taxing grip
- Heavy compound push/pull only when arms feel recovered from gaming
- Never skip arms — skip legs before arms given current aesthetic goal
- On commute days: banded work or bodyweight at friend's place

## Current Split Template
| Day | Session | Notes |
|-----|---------|-------|
| Mon | Legs (heavy) | Squats, leg press, RDLs, calves |
| Tue | Arms + shoulders (pump) | Curls, pushdowns, lateral raises. 15-20 reps. |
| Wed | Rest / gaming | Stretch forearms |
| Thu | Legs + arms | Legs moderate, arm supersets |
| Fri | Push or Pull (if arms allow) | If fried → legs again |
| Sat | Arms + shoulders (pump) | Same as Tue |
| Sun | Rest | |

## Key Metrics to Track
- Can he grip the bar without forearm pain?
- Are arms recovering between gaming sessions?
- Is he making it to the gym 4+ times per week?
