# Supplement & Pharmacology Stack

## Active Stack (as of May 2026)

### Morning/Afternoon
- Caffeine + L-Theanine 200mg (smoother stimulation, less vasoconstriction)
- Creatine 5g (non-negotiable — biggest visual impact for fullness)
- Vitamin D3 5000 IU (Ireland + indoors = deficient)
- Fish Oil 2-3g EPA+DHA (anti-inflammatory, BP, vascular function)

### Pre-Training
- L-Citrulline 6-8g (vasodilation, pumps, offsets caffeine constriction)
- Tadalafil 5mg daily (vascularity, pumps, forearm blood flow, offsets caffeine BP effects)

### Before Bed
- Magnesium Glycinate 400mg (sleep quality, muscle relaxation, likely deficient)
- Taurine 2g (mild vasodilator, calms CNS, supports heart)
- Ashwagandha KSM-66 600mg (cortisol reduction, anxiety, recovery)

## Considered / Conditional
| Compound | Status | Rationale |
|----------|--------|-----------|
| BPC-157 250-500mcg/day | Conditional | Only if forearm strain blocks consistent training |
| TB-500 | Skipped | Overkill for chronic strain |
| MK-677 / GH secretagogues | Skipped | Water retention + 4 weeks too short |
| Retatrutide | Rejected | Not warranted at 62.8kg, would accelerate muscle loss |
| CoQ10 100-200mg | Optional | If chronic caffeine use continues long-term |
| Rhodiola Rosea | Optional | Anti-fatigue for long gaming sessions |
| Garlic extract / Kyolic | Optional | Gentle BP reduction, stacks with everything |
| Low-dose melatonin 0.3-0.5mg | Optional | Only if shifting sleep earlier is attempted |

## Sourcing (Ireland)
- Tadalafil: Zavamed.com, LloydsPharmacy Online Doctor, or GP (ask for generic 5mg)
- BPC-157: Research chemical vendors (unregulated quality — caveat emptor)
- Everything else: Health stores, bulk powders, Amazon

## Key Interactions
- Caffeine + Tadalafil: complementary — tadalafil offsets caffeine's vasoconstriction
- Caffeine + L-Theanine: synergistic — smooth energy without jitters
- Magnesium + sleep: improves deep sleep percentage
- Creatine + carbs: enhanced uptake when taken with carbohydrates

## Rules
- Last caffeine by 4-5pm (half-life ~5-6hrs affects deep sleep)
- Estimated monthly cost: €30-40 for core stack
