# 3-Week Arm Priority Program (June 4 - June 26)

## Philosophy
3 guaranteed sessions/week. Arms every session. Muscle memory + creatine = rapid visual change.
Cables/machines for arms to avoid competing with gaming fatigue.
If a 4th/5th day happens, just do arms + whatever.

## Schedule: 3 days/week (e.g. Tue/Thu/Sat or Mon/Wed/Fri)

---

## Day 1: Push + Arms

**Compound:**
- Incline DB Press: 4 x 8-10
- Dips or Machine Press: 3 x 10-12

**Triceps:**
- Cable Rope Pushdowns: 4 x 12-15
- Overhead Cable Extension: 3 x 12-15

**Biceps:**
- Incline DB Curls: 3 x 10-12
- Hammer Curls: 3 x 10-12

---

## Day 2: Pull + Arms

**Compound:**
- Pull-ups or Lat Pulldown: 3 x 8-10
- Cable Row: 3 x 10-12

**Biceps:**
- Barbell Curls: 4 x 8-10
- Spider Curls: 3 x 10-12

**Triceps:**
- Rope Pushdowns: 3 x 12-15
- Close-Grip Push-ups: 2 x failure

---

## Day 3: Arms + Shoulders

**Shoulders:**
- DB Shoulder Press: 3 x 10-12
- Lateral Raises: 4 x 12-15

**Biceps:**
- EZ Bar Curl: 4 x 8-10
- Concentration Curls: 3 x 10-12

**Triceps:**
- Skull Crushers: 4 x 10-12
- Cable Kickbacks: 3 x 12-15

---

## Progression
- Complete all reps at top of range → increase weight next session
- Muscle memory = strength jumps fast in weeks 1-2
- At least 1 day rest between sessions

## Abs (optional, end of any session)
- Hanging Leg Raises: 3 x 10-12

## If gaming fatigue is severe
- Skip grip-heavy movements (barbell curls, heavy rows)
- Replace with cables/machines that session
