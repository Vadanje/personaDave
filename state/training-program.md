# 4-Week Arm Priority Program (June 3 - June 26)

## Philosophy
Every session ends with direct arm work. Compound movements maintain overall stimulus.
Gaming fatigue means avoiding heavy grip work (deadlifts, barbell rows, heavy shrugs).
Cables and machines preferred for arms - constant tension, no grip limitation.

## Schedule: 5 days/week
- Day A: Push + Arms
- Day B: Pull + Arms
- Day C: Arms + Shoulders
- Day D: Push + Arms
- Day E: Pull + Arms
- Rest: 2 days (flexible placement around gaming/commute)

---

## Day A: Push + Arms

**Compound:**
- Incline DB Press: 4 x 8-10
- Flat Machine Press or Dips: 3 x 10-12

**Triceps (direct):**
- Cable Rope Pushdowns: 4 x 12-15
- Overhead Cable Extension: 3 x 12-15
- Close-Grip Push-ups (burnout): 2 x failure

**Biceps (direct):**
- Incline DB Curls: 3 x 10-12
- Cable Hammer Curls: 3 x 12-15

---

## Day B: Pull + Arms

**Compound:**
- Pull-ups: 3 x 8-10
- Cable Row (neutral grip): 3 x 10-12

**Biceps (direct):**
- Barbell Curls: 4 x 8-10
- Spider Curls (incline bench): 3 x 10-12
- Cable Curl (single arm): 3 x 12-15

**Triceps (direct):**
- Dip Machine or Bench Dips: 3 x 10-12
- Single Arm Cable Pushdown: 3 x 12-15

---

## Day C: Arms + Shoulders

**Shoulders:**
- DB Shoulder Press: 3 x 10-12
- Lateral Raises: 4 x 12-15

**Biceps:**
- EZ Bar Curl: 4 x 8-10
- Concentration Curls: 3 x 10-12
- Reverse Curls (forearm thickness): 3 x 12

**Triceps:**
- Skull Crushers or Overhead DB Extension: 4 x 10-12
- Cable Kickbacks: 3 x 12-15
- Diamond Push-ups: 2 x failure

---

## Day D: Push + Arms

**Compound:**
- DB Bench Press (flat): 4 x 8-10
- Cable Flyes: 3 x 12-15

**Triceps (direct):**
- V-Bar Pushdowns: 4 x 10-12
- Overhead Rope Extension: 3 x 12-15
- Close-Grip Smith Press: 3 x 8-10

**Biceps (direct):**
- Preacher Curls (machine or EZ bar): 3 x 10-12
- Hammer Curls: 3 x 10-12

---

## Day E: Pull + Arms

**Compound:**
- Lat Pulldown: 3 x 10-12
- Seated Cable Row: 3 x 10-12

**Biceps (direct):**
- Cable Curls (straight bar): 4 x 12-15
- Incline DB Curls: 3 x 10-12
- 21s (barbell or EZ bar): 2 sets

**Triceps (direct):**
- Rope Pushdowns: 3 x 12-15
- Overhead DB Extension (single arm): 3 x 10-12

---

## Progression Strategy
- Week 1-2: Focus on form, mind-muscle connection, moderate weight
- Week 3-4: Push intensity, add weight or reps each session
- Every session should have 1-2 sets taken to failure on isolation work

## Abs (2-3x/week, end of session)
- Hanging Leg Raises: 3 x 10-12
- Cable Crunches: 3 x 12-15

## Notes
- Gym closes 9:30pm - aim to start by 8pm latest
- Pre-workout 30min before session
- If arms still sore from previous day, shift to higher rep/lighter weight rather than skipping
- If gaming fatigue is severe in forearms, skip grip-heavy movements that day
